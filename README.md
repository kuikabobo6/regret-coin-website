# $REGRET Token Website

Static website for $REGRET - The Crypto Regret Token

## How to Deploy

1. Create GitHub account
2. Create new repository: `regret-coin-website`
3. Upload `index.html`
4. Go to Settings > Pages
5. Select "Deploy from branch" > main branch
6. Save - your site will be at: `username.github.io/regret-coin-website`

## Updates Needed

1. Replace `[CONTRACT_ADDRESS_WILL_BE_HERE]` with real contract
2. Update launch date in countdown
3. Add real Telegram/X links when available
4. Customize colors if desired
